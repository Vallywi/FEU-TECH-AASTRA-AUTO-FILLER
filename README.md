**Privacy Policy for FEU Tech AASTRA Survey Auto-Filler**

The FEU Tech AASTRA Survey Auto-Filler Chrome extension is designed to assist users in completing survey forms more efficiently by automatically filling in predefined responses.

This extension does not collect, store, transmit, or share any personal or sensitive user data. All actions performed by the extension occur locally within the user’s browser and are limited to interacting with the active AASTRA survey page when triggered by the user.

The extension does not track user activity, does not use cookies, and does not connect to external servers for data processing. It only accesses the necessary webpage elements to perform its intended function of auto-filling survey responses.

We are committed to protecting user privacy and ensuring a safe browsing experience. If you have any questions or concerns regarding this policy, you may contact the developer through the Chrome Web Store listing.
